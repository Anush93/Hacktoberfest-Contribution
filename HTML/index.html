Ikutin Tutorial Di Bawah Ini





Untuk menginstal php-pastebin, Anda tidak perlu banyak!
1.Server web

2. Database MySQL

3. mod_rewrite diaktifkan di server web Apache Anda (lihat di phpinfo() Anda) Untuk memulai,
kirim file Anda di ftp dan buka browser Anda ke nama domain yang dipilih,

Anda akan melihat halaman instalasi. Ikuti petunjuk penginstalan.




Setelah instalasi selesai, Anda harus menghapus file install.php, libs/db.sql dan meletakkan chmod 644 pada file libs/db.php




Untuk memperbarui versi php-pastebin Anda, Anda harus:
1. Hapus versi lama Simpan file berharga libs/db.php (yang berisi sql login Anda)
Hapus semua file FTP Anda 2. Unggah versi baru
Unggah v3 php-pastebin di FTP Anda

Ganti db.php dengan Anda 3. Unduh




dan tambahkan di root script Anda * Buka navigator Anda dan buka upgrade.php * Klik "Tingkatkan v2 ke v3 sekarang!" 4. Hapus upgrade.php dan buka situs web Anda
